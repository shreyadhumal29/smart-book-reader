# 📚 Smart Book Reader

## 🚀 Project Overview

Smart Book Reader is a web-based application that allows users to explore books, search content, and interact with a simple AI chatbot for quick answers.

This project is built using HTML, CSS, and JavaScript and demonstrates a basic implementation of an online book reader with chatbot functionality.

---

## ✨ Features

* 📖 Online Book Reader (PDF support)
* 🔍 Search functionality for books
* 🤖 AI Chatbot (keyword-based)
* 🎨 Simple and clean user interface
* 📱 Responsive design (works on different devices)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## ▶️ How to Run

4. Open in browser
